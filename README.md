# GitHub Pages Starter Template

A simple website template for learning Git and GitHub Pages.

## Files

- `index.html` - Main webpage
- `style.css` - Styling
- `script.js` - Interactive features
- `instructions.md` - Step-by-step guide for students

## Quick Start

1. Clone this repository
2. Follow the instructions in `instructions.md`
3. Push to your own GitHub repository
4. Enable GitHub Pages in Settings
5. Your site will be live at `https://[username].github.io/[repository-name]`

## For Students

Open `instructions.md` for detailed step-by-step instructions on how to:
- Set up Git and GitHub
- Create your own repository
- Publish your website with GitHub Pages
